# CraftingSwift-SampleProjects

A collection of sample projects I use for my YouTube channel. https://youtube.com/@craftingswift

And also on my blog https://craftingswift.dev/articles

## Project Index

- [HabitsWidget](./HabitsWidget): A WidgetKit sample extracted from a personal app that shows today's pending habits from SwiftData and includes a few layout-debug helpers. Related: 
    - [iOS Widget UI Design and Implementation in SwiftUI](https://www.youtube.com/watch?v=511OIgHz-sg).
- [ImageGallery](./ImageGallery): A macOS sample that imports folders with security-scoped bookmarks, keeps access to sandboxed locations, and renders the images in a split-view gallery. Related: 
    - [Making an app that manages files in macOS with the app sandbox](https://www.youtube.com/watch?v=zz8HB_ugM5E).
- [LiveActivityDemo](./LiveActivityDemo): A match-score Live Activity demo built with ActivityKit, WidgetKit, reusable UI packages, and helpers for push-to-start and remote updates. Related: 
    - [Design and develop Live Activities with ActivityKit for iOS 18](https://www.youtube.com/watch?v=mu9LlmUYC9E).
- [MovieCatalog](./MovieCatalog): A SwiftUI movie browsing app used to explore previews and testing, navigation, and modularization with a more production-style structure. Related: 
    - [Building UI that is easy to Preview and Test with SwiftUI](https://www.youtube.com/watch?v=KZBr0wlS3K0)
    - [Advanced Navigation for SwiftUI apps](https://www.youtube.com/watch?v=Z-3ETLYbhFA)
    - [How to modularize an iOS app with SPM packages](https://www.youtube.com/watch?v=IjoXs5aeQ70).
- [QuickPrototypes](./QuickPrototypes/README.md): A scratchpad app for UI experiments, currently focused on recreating Slack's iOS navigation bar in SwiftUI. 
    - [Recreating Slack's iOS Navigation Bar in SwiftUI](https://www.youtube.com/watch?v=I7lUqCw5S8g).
- [TrackingEvents](TrackingEvents/README.md): A YAML-driven analytics event definition setup with a generator that produces strongly typed iOS tracking APIs. 
    - [syncing-ios-and-android-tracking-events](https://craftingswift.dev/articles/syncing-ios-and-android-tracking-events).
- [UIKitNavigation](UIKitNavigation/README.md): UIKit examples for styling `UINavigationBar` with large titles, custom title views, segmented controls, and transparent scroll-edge appearances. Related: 
    - [Styling the navigation bar on iOS with UIKit](https://www.youtube.com/watch?v=NtmA7o3hmhI).
