# Live Activity Demo

This sample shows how to build and test a match-score Live Activity with
ActivityKit and WidgetKit. It includes the app target, reusable content and UI
packages, and helpers for push-to-start and remote push updates.

## Tools

[Push Notification Console](https://icloud.developer.apple.com/dashboard/notifications)
is useful when testing live activity pushes.
