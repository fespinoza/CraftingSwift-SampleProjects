# Habit Widget

This is some sample code extracted from a personal project (private repo), so
the code won't compile, but you can check it out and quickly try to replicate
something similar in your project.

The sample focuses on a WidgetKit widget that shows today's pending habits,
loads data from SwiftData, and includes some small debug helpers that make
widget layout work easier.
