# Quick Prototypes

This folder is where I keep small SwiftUI spikes and UI experiments before
deciding whether an idea should become a fuller sample project.

The current checked-in prototype focuses on recreating Slack's iOS navigation
bar in SwiftUI and comparing a few different approaches for the same effect.

## Included Prototype

- **SlackNavBar**: experiments with ZStack, opacity-based, and view-modifier
  implementations of the same custom navigation bar effect.
